Web Applications
================

These scripts focus on web-facing caching patterns such as cache-aside,
HTTP response caching, and invalidation.

Relevant Scripts
----------------

- ``examples/getting_started/cache_aside_pattern.py``
- ``examples/getting_started/http_api_response_cache.py``
- ``examples/getting_started/db_query_cache_with_invalidation.py``
- ``examples/advanced/rate_limit_with_cache.py``
- ``examples/advanced/feature_flags_cached.py``
- ``examples/advanced/graceful_degradation.py``
- ``examples/advanced/stampede_protection.py``

Run Examples
------------

.. code-block:: bash

   python examples/getting_started/cache_aside_pattern.py
   python examples/getting_started/http_api_response_cache.py
   python examples/getting_started/db_query_cache_with_invalidation.py
   python examples/advanced/rate_limit_with_cache.py
