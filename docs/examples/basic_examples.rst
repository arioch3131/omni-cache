Basic Examples
==============

These examples are the fastest way to learn omni-cache usage patterns.

Getting Started Scripts
-----------------------

- ``examples/getting_started/basic_usage.py``
- ``examples/getting_started/basic_memory_adapter.py``
- ``examples/getting_started/basic_redis_adapter.py``
- ``examples/getting_started/basic_async_redis_adapter.py``
- ``examples/getting_started/basic_memcached_adapter.py``
- ``examples/getting_started/basic_smartpool_adapter.py``
- ``examples/getting_started/custom_file_cache_adapter.py``
- ``examples/getting_started/cache_aside_pattern.py``
- ``examples/getting_started/http_api_response_cache.py``
- ``examples/getting_started/db_query_cache_with_invalidation.py``

Run Examples
------------

.. code-block:: bash

   python examples/getting_started/basic_usage.py
   python examples/getting_started/basic_memory_adapter.py
   python examples/getting_started/basic_redis_adapter.py
   python examples/getting_started/basic_async_redis_adapter.py

Recommended Order
-----------------

1. ``basic_usage.py``
2. ``basic_memory_adapter.py``
3. ``basic_redis_adapter.py``
4. ``basic_async_redis_adapter.py``
5. ``basic_memcached_adapter.py``
6. ``basic_smartpool_adapter.py``
